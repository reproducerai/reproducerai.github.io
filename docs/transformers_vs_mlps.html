Experiment: MLP vs Self-Attention+MLP on Sequence Data

Objective

Compare an MLP and Self-Attention+MLP for a simple sequence classification task.

Dataset

  • Definition

    Sequences of length 10, 20, 30, …, up to 100, each consisting of 8-dimensional vectors drawn from a normal distribution (mean = 0, std = 1). About half of the sequences (random 50%) contain 3 consecutive embeddings from a different normal distribution (mean = 1, std = 0.5 + tiny noise) inserted at a random position. Sequences containing this 3-vector pattern are labeled 1; those without it are labeled 0.

  • Shape and Distribution

    • Train: Shape 10000 × sequence_length × 8, Positive/Negative: (4969, 5031)
    • Validation: Shape 1500 × sequence_length × 8, Positive/Negative: (733, 767)
    • Test: Shape 1500 × sequence_length × 8, Positive/Negative: (782, 718)

Models

  • MLP (Multi-Layer Perceptron): 53,761 trainable parameters
  • Self-Attention + MLP: 54,049 trainable parameters

Training

  • Task: Binary classification.
  • Optimizer: Adam, initial learning rate = 1e-4, all other settings default.
  • Loss: BCEWithLogitsLoss, default settings.
  • Training duration: 25 epochs.

Findings

  • For short sequences (seq_len = 10), both MLP and Self-Attention+MLP achieve high accuracy.
  • For longer sequences (seq_len = 100), Self-Attention+MLP outperforms MLP significantly.

Conclusion

  • MLPs work well for short sequences but fail as sequences grow longer and sparser.
  • Self-Attention+MLP handles long sequences effectively by capturing sequence structure and focusing on relevant positions.
