Experiment: MLP vs Self-Attention+MLP on Sequence Data

Objective

Compare an MLP and Self-Attention+MLP for a simple sequence classification task.

Dataset

  • Definition

    We generate sequences of length S, where each element is an 8-dimensional vector. For approximately half of the sequences (randomly selected 50%), we insert three consecutive vectors drawn from a normal distribution (mean = 1, std = 0.5 + small noise) at a random position within the sequence. The remaining S − 3 vectors in those sequences are sampled from a standard normal distribution (mean = 0, std = 1). For the other half (without the inserted pattern), all S vectors are sampled from the standard normal distribution. Sequences containing the 3-vector pattern are labeled 1, and sequences without it are labeled 0.

  • Shape and Distribution

    • Train: Shape 10000 × S × 8, Count 1/0: (4969, 5031)
    • Validation: Shape 1500 × S × 8, Count 1/0: (733, 767)
    • Test: Shape 1500 × S × 8, Count 1/0: (782, 718)

Models

  • MLP (Multi-Layer Perceptron): 53,761 trainable parameters
  • Self-Attention + MLP: 54,049 trainable parameters

Training

  • Task: Binary classification.
  • Optimizer: Adam, initial learning rate = 1e-4, all other settings default.
  • Loss: BCEWithLogitsLoss, default settings.
  • Training duration: 25 epochs.

Findings

  • For short sequences, both MLP and Self-Attention+MLP achieve high accuracy.
  • For longer sequences, Self-Attention+MLP outperforms MLP significantly.

Conclusion and Code

  • MLPs work well for short sequences but fail as sequences grow longer.
  • Self-Attention+MLP handles long sequences effectively by capturing sequence structure and focusing on relevant positions.
